# Project_monitoring-ai
Fullstack Project Management with overall Construction buisiness overview just in this web app where User Auth + Multi-Project Dashboard, Role-Based Access, AI integrated, live data automation each dashboard, Financial Control, liability, Inventory , DPR, Docs auto deep scan &amp; live automation, in a word fully professional best construction PM tool.
# BuildTrack AI - Fullstack Construction Project Management

Real JWT Auth + Per-Project RBAC + AI Automation + MongoDB

## Quick Start
1. Clone this repo
2. cd backend && npm install && cp .env.example .env
3. cd ../frontend && npm install && cp .env.example .env
4. Fill your MONGO_URI + GEMINI_API_KEY + JWT_SECRET
5. docker-compose up --build

Frontend deploys on Vercel, Backend on Render.
